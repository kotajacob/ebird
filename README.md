# EBIRD

TUI Flappy Bird. It's a lil bit jank tbh.

![ebird screenshot](ebird.webp)

## Build and Install

Build dependencies:
 * golang
 * make

Optionally configure `config.mk` to specify a different install location.
Defaults to `/usr/local/`

`sudo make install`

## Uninstall

`sudo make uninstall`
